If the model state is valid and the records are created successfully, redirects to the Create action of the Trykk controller. If no last service order is found, adds a model error. If the model state is not valid, re-displays the view with the current viewModel and error messages.
Remarks
The method ensures that the first FunksjonsTest in the list carries the comment over from the view model. A success message is displayed after the records are saved.
