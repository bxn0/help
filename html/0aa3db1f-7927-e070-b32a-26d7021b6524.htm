 NameDescription
Register Initiates the user registration process. This method checks if the 'Admin' and 'Ansatt' roles exist and creates them if they do not. It then presents a registration view model to the user.
Register(RegisterViewModel) Processes the user registration submission. If the model state is valid, it creates a new user and assigns a role based on the submitted data. On successful registration, the user is redirected to the dashboard. If the registration fails, it returns the registration view with error messages.
