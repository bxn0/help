Parameters

id  NullableInt32
The ID of the Hydrolisk entity to find. If null, the method returns a NotFound result.

Return Value

TaskIActionResult
A task that represents the asynchronous operation. The task result contains the IActionResult for the Details view. If the entity with the specified ID is not found, NotFound (HTTP 404) is returned. Otherwise, the Details view for the Hydrolisk entity is displayed.
