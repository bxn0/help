Hydrolisk Properties

The Hydrolisk type exposes the following members.

Properties
 NameDescription
BorSkiftes Gets or sets a value indicating whether the item should be replaced. If "Bør skiftes" is selected, this will be set to true.
ChecklistItem Gets or sets the checklist item description.
Defekt Gets or sets a value indicating whether the item is defective. If "Defekt" is selected, this will be set to true.
Id Gets or sets the identifier for the Hydrolisk object.
Kommentar Gets or sets comments related to the checklist item. This is a free-text field for additional notes or comments.
OK Gets or sets a value indicating whether the checklist item is marked as OK. If "OK" is selected, this will be set to true.
Ordrenummer Gets or sets the order number associated with the service order.
ServiceOrder Navigation property for the associated ServiceOrder. This allows for lazy loading of the service order related to this Hydrolisk object.
ServiceOrderId Gets or sets the service order identifier. This serves as a foreign key to the ServiceOrder table.
