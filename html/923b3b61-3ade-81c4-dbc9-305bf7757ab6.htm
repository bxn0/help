Parameters

serviceOrderId  Int32
The ID of the service order to pre-populate the new Trykk entity form.
category  String  (Optional)
The category of the service order, with a default value of "Trykk".

Return Value

IActionResult
If the service order is not found, NotFound (HTTP 404) is returned. Otherwise, the Create view for a Trykk entity is displayed, pre-populated with data from the service order.
Remarks
If there is a success message stored in TempData, it will be displayed using the notification service.
