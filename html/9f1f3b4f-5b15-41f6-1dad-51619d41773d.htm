AccountControllerForgotPassword(ForgotPasswordViewModel) Method

Processes the password reset request. If the model state is valid and the user exists, a password reset token is generated and a link is sent to the user's email. Redirects to 'ForgotPasswordConfirmation' view upon completion, regardless of whether the user was found.
